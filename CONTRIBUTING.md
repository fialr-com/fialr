# Contributing

fialr is proprietary software. Source code is not distributed in this repository. This repo serves as the public issue tracker.

## Reporting issues

- [Report a bug](https://github.com/fialr-com/fialr/issues/new?template=bug_report.yml)
- [Request a feature](https://github.com/fialr-com/fialr/issues/new?template=feature_request.yml)

For security vulnerabilities, do not open a public issue. Email security@fialr.com.

For billing and licensing issues, email licensing@fialr.com.

## Repository configuration

This repository belongs to the `fialr-com` GitHub account.

**Git identity:**

```
git config user.name "fialr-com"
git config user.email "267184569+fialr-com@users.noreply.github.com"
```

**Remote:**

```
origin  git@github-fialr:fialr-com/fialr.git
```

The `github-fialr` host alias is defined in `~/.ssh/config` and resolves to `github.com` with the SSH key for the `fialr-com` account. All commits to this repository must use the identity above.
